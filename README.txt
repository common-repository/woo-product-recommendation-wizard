== Description ==

Woo Product Recommendation Wizard let customers narrow down the product list on the basis of their choices. It enables the store owners to add a questionnaire to the product page. The product recommendations are then rendered according to the answers, given by the users. You can showcase 'n' number of products, matching the answers and query. 

Plugin Demo: <a href="http://wooproductrecommendationwizard.demo.store.multidots.com"> View Demo </a>

= HOW IT WORKS? =
1. Before add wizard you will have to select product attribute in particular product.
2. Enter Wizard title, and enable status. 
3. Manage Question: Add New Question( Create question option like a radio button ) 
4. Copy and past Wizard shortcode in page/ post for show wizard.

= Key Features =

* Create Wizard (Questions and answers) 
* Manage Question: Add New Question 
* Manage Options: Add New Attribute Name and value.
